# 古今抄

読み継がれるには、理由がある。

## 変更点 v3

縦スワイプを主操作から外しました。
理由は、本文・超訳・基本訳・読み解きメモを読むUIと縦スワイプが競合し、スマホで使いづらかったためです。

現在は下部固定ナビの「前へ」「次へ」を主操作にしています。
PCではホイールと上下キーも使えます。
