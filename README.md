# 古今掌

小学生にもわかる、古典のいい言葉。  
親子で読める古典カードアプリです。

Developed by memetomamoru / 芽芽斗守  
https://note.com/memetomamoru

## 収録内容

- 百人一首100件
- 論語60件
- 故事成語40件
- ことわざ50件
- 慣用句50件
- 合計300件

外部JSONは `data/kokonsho_300.json` を参照します。  
HTML内蔵JSONと旧互換データは使っていません。

## v100

v99を起点に、継続利用のための「まいにち10枚」と「読んだきろく」を追加しました。

- ホーム上部に「まいにち」導線を追加
- 300件から日付ごとに10枚を自動選出
- 同じ日付では同じ10枚を表示
- 各カードに「読んだ」チェックを追加
- 日付ごとに読んだカードを見られる「読んだきろく」を追加
- 「すき」保存は従来どおり利用可能
- 読んだ記録は端末内のlocalStorageに保存
- 保護者ページとプライバシーポリシーに、読んだきろくの保存説明を追加

## 保存キー運用

ユーザー記録を上書きリリースで消さないため、以下のlocalStorageキーを固定します。

- `kokonsho_favorites_v1`：すき保存
- `kokonsho_daily_sets_v1`：日付ごとの10枚
- `kokonsho_read_log_v1`：読んだ記録
- `kokonsho_daily_progress_v1`：進捗

アプリ本体のバージョンがv101以降になっても、保存形式を変えない限りキー名は変更しません。  
ユーザー記録をまとめて消す初期化処理や、該当キーを削除する処理は使わない方針です。
