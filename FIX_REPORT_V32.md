# 古今掌 v32 妖精アプリUI刷新レポート

新しい正ファイル:
- `kokonsho_github_ready_package_v32_fairy_app_rebuild.zip`

## 今回の目的
画像生成で固めた方向性に合わせて、古今掌を「Webサイト」ではなく「毎日ひらく親子向けアプリ」に見えるよう刷新しました。

## 主な変更

### 1. 妖精マスコットを主役化
- 生成済みキャラクター画像から妖精だけを切り出し、`assets/images/fairy_mascot.webp` として追加
- ホーム、さがす、すき、クイズの各画面で案内役として表示
- 現代の子どもキャラは使わず、妖精をサービスの人格に寄せました

### 2. 4タブ構成のアプリUIに刷新
- ホーム
- さがす
- すき
- クイズ

下部固定タブバーで切り替える構成にし、Webページ感を減らしました。

### 3. こども訳ファーストを維持しつつ再設計
- ホームの主役を「今日のことば」カードに変更
- 最初に大きく「こども訳」を表示
- その下に 原文 / 読み / ことばポイント / 親子で話してみよう を整理

### 4. ふりがな表示を追加
- 主要な漢字語に `<ruby>` でふりがなを付与
- こども訳、原文、ことばポイント、親子の問い、一覧カードなどに反映
- 完全な全文自動かな変換ではなく、主要語彙に対する実装です

### 5. 色数を整理
- 生成り
- 若草色
- うすい金
- やわらかい朱色
- うすい空色

上記を基調にして、以前より統一感を出しました。

### 6. OGPとアプリアイコンを更新
- `assets/images/ogp.png` を親子向け・妖精主役の内容に再作成
- アイコンも古今掌の世界観に合わせて再描画

## 変更していないもの

- 外部JSON: `data/kokonsho_160.json`
