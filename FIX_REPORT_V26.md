# 古今抄 v26 修正内容レポート

新しい正ファイル:
- `kokonsho_github_ready_package_v26_supertranslation_typography.zip`

今回の主目的:
- 超訳の見せ方を、見た瞬間に「ここが主役」と分かる方向へ強化する
- 原文・基本訳との役割差を、フォント・余白・装飾・情報設計で明確化する

## 実施内容

### 1. 超訳エリアのタイポグラフィ再設計
- `#transBox` に専用ヘッダーを追加
- 小さなインラインSVGバッジを追加
- 超訳時は、より大きく・太く・余白を広くした表示に変更
- 基本訳時は、落ち着いた補助テキストとして見えるよう抑制
- 行長を少し制御し、超訳本文が塊として読みやすく見えるよう調整

### 2. 視覚階層の強化
- 超訳ラベルを小さな補助情報化し、本文を主役化
- 原文・読みは引き続き明朝系で静かな見え方を維持
- 基本訳表示時は、ヘッダー表記も `基本訳` に切り替わるよう変更

### 3. 装飾方針
- 長文本文をSVGテキストにするのではなく、装飾だけにSVGを使う方針を採用
- 超訳本文は通常テキストのままにし、可読性・レスポンシブ性・保守性を優先

## 変更ファイル
- `index.html`

## 変更していないもの
- データ内容（160件、toneType、表記）
- PC背景画像
- スマホ背景非表示仕様
- コピーなし、縦スワイプなし、ホイール送りなし
- 一覧、検索、ジャンル自動生成、ランダム仕様

## 補足
- 外部Webフォントの新規追加は今回見送り
- 理由は、日本語Webフォントは容量が大きくなりやすく、表示速度やFCPへの影響があるため
- まずは現行のローカルフォント環境を活かしつつ、見せ方の設計で超訳を強くする方針を採用
